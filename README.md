# FMCW Radar Simulation

![FMCW Radar](https://www.google.com/url?sa=i&url=https%3A%2F%2Fen.wikipedia.org%2Fwiki%2FContinuous-wave_radar&psig=AOvVaw0RYc9byHOmboGXPU1Ep4vy&ust=1693840207315000&source=images&cd=vfe&opi=89978449&ved=0CBAQjRxqFwoTCMDr39PcjoEDFQAAAAAdAAAAABAE)

## Overview

This repository contains a Python-based FMCW (Frequency Modulated Continuous Wave) radar simulation. FMCW radar is a key technology used in various applications, including remote sensing, object detection, and navigation systems.

## Features

- **Signal Generation:** Simulate FMCW radar signals with customizable parameters.
- **Target Simulation:** Simulate target objects with distance and radial velocity.
- **Signal Processing:** Implement signal processing techniques, including demodulation and range estimation.

## Dependencies

1. numpy
2. matplotlib
3. scipy
